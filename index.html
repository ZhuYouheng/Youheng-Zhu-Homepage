Fluid Simulation: Lid-Driven Cavity Flow

Lid-driven cavity flow is a classical benchmark problem for CFD. In this scenerial, the fluid is placed in a 2-Dimensional box, and the lip on the top of the box is moving at a fixed velocity. The fluid inside the box is then driven by the movement of the lip before it stabilizes.

The videos below shows the evolution of the velovity field with regards to the iteration count using the Lattice-Boltzmann Method (LBM). The task was implemented using taichi. The source of the code can be found at simple-LBM-based-on-taichi.
